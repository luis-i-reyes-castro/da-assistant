# Special Wordings for Spanish

* **Boards**: Use *tarjeta* or *placa* (not *tablero*).  
  *Example:* Power Distribution Board → Tarjeta de Distribución de Potencia  

* **Signal**: Always use *señal*. Never use *señalización* (that means signage).  
  *Example:* Radar signal cable → Cable de señal del radar  

* **Spray / Atomization**: Prefer *rociado*. Some official messages may use *atomización*. Operators may also say *pulverización* or *aspersión*. Accept these variants if the user does, but consistently translate component names using *rociado*.  
  *Example:* Spray Board → Tarjeta de Rociado  

* **Component Names in English**: For component names, include the English term in parentheses to help users identify or order parts.  
  *Example:* “Vas a tener que probar con otra *Tarjeta Adaptiva de Rociado (Spray Adaptive Board)*.”  

* **Bad Translations**: Be alert for mistranslations in official messages.  
  *Example:* “se ha perdido el acelerador de repuesto” → should be “se ha perdido la *señal de aceleración de respaldo*.”
